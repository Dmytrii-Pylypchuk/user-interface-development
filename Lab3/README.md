# Text Editor (Qt + Python)

Лабораторна робота з розробки графічного інтерфейсу користувача з використанням Qt.

## Опис

Цей проєкт реалізує простий **текстовий редактор**, який дозволяє переглядати та редагувати текстові файли.
Інтерфейс користувача створений за допомогою Qt та реалізований мовою Python.

Застосунок містить стандартні функції роботи з файлами та відповідає базовим принципам якості інтерфейсу користувача.

## Використані технології

* Python
* Qt Framework
* PyQt5

## Основні можливості

* створення нового текстового файлу
* відкриття існуючого файлу
* редагування тексту
* збереження файлу
* збереження файлу під новим ім’ям
* підтвердження збереження при наявності незбережених змін
* підтвердження перед закриттям програми
* гарячі клавіші для основних операцій
* відображення імені файлу у заголовку вікна
* status bar з інформаційними повідомленнями

## Гарячі клавіші

| Команда       | Клавіші          |
| ------------- | ---------------- |
| Новий файл    | Ctrl + N         |
| Відкрити файл | Ctrl + O         |
| Зберегти файл | Ctrl + S         |
| Зберегти як   | Ctrl + Shift + S |
| Вийти         | Ctrl + Q         |

## Функціональність інтерфейсу

Меню **Файл** містить такі команди:

* **Новий** – створення нового файлу
* **Відкрити** – відкриття існуючого текстового файлу
* **Зберегти** – збереження поточного файлу
* **Зберегти як** – збереження файлу з новим ім’ям
* **Вийти** – завершення роботи програми

Якщо у файлі є незбережені зміни, програма пропонує користувачу:

* зберегти зміни
* закрити без збереження
* скасувати дію
