# user-interface-development
## Назва дисципліни:
# Розробка інтерфейсів користувача

# Назва роботи №1:
Розробка застосунку "Текстовий редактор"

# Опис:
Даний проєкт виконано в межах навчальної дисципліни
«Розробка інтерфейсів користувача».
Метою роботи є створення настільного застосунку,
який дозволяє переглядати та редагувати текстові файли
за допомогою графічного інтерфейсу користувача.

# Функціональні можливості:
- створення нового текстового файлу;
- відкриття існуючих текстових файлів формату .txt;
- редагування текстового вмісту;
- збереження змін у файл;
- вихід з програми через меню.

# Використані технології:
- мова програмування C#;
- платформа .NET;
- технологія Windows Forms;
- технологія Windows Presentation Foundation (WPF);
- середовище розробки Visual Studio.

# Структура проєкту:
Проєкт містить окремі реалізації текстового редактора
з використанням Windows Forms та WPF.
Кожна реалізація має власну структуру файлів та інтерфейс.

# Запуск проєкту:
1. Відкрити рішення у Visual Studio.
2. Обрати необхідний проєкт (WinForms або WPF).
3. Запустити застосунок (клавіша F5)
